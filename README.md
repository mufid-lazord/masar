# تطبيق مَسَار (MASAR)

تطبيق جوال لتقديم الخدمات الأساسية للفلسطينيين المقيمين في مصر - مشروع تخرج.

## 📱 وصف التطبيق

تطبيق مَسَار هو منصة موحدة وموثوقة تجمع معلومات دقيقة ومحدثة عن الخدمات الأساسية اللي يحتاجها الفلسطينيون في حياتهم اليومية في مصر. يعالج التطبيق مشكلة تشتت المعلومات ويوفر بديلاً موثوقاً عن مجموعات السوشيال ميديا غير الرسمية.

## 🎯 المميزات الرئيسية

- **12 شاشة احترافية** مع دعم كامل للغة العربية واتجاه RTL
- **4 أقسام رئيسية للخدمات:**
  - 🏠 السكن (شقق، استوديوهات، سكن طلاب)
  - ⚕️ الرعاية الصحية (مستشفيات، عيادات، صيدليات)
  - 🎓 التعليم (جامعات، مدارس، معاهد)
  - 🏛️ السياحة (معالم أثرية وأماكن سياحية)
- **16 خدمة جاهزة** مع تفاصيل كاملة
- **خريطة تفاعلية** لعرض الخدمات حسب الموقع
- **بحث متقدم** مع فلاتر وتاريخ البحث
- **ملف شخصي** مع إعدادات كاملة

## 🛠️ التقنيات المستخدمة

- **Flutter 3.x** - إطار تطوير التطبيق
- **Dart** - لغة البرمجة
- **google_fonts** - خط Cairo العربي
- **flutter_localizations** - دعم اللغة العربية
- **shared_preferences** - حفظ الإعدادات المحلية
- **cached_network_image** - تحميل الصور

## 🚀 خطوات التشغيل

### 1. المتطلبات

- Flutter SDK (إصدار 3.0 أو أحدث)
- Dart SDK (إصدار 3.0 أو أحدث)
- Android Studio / VS Code
- محاكي Android أو جهاز فعلي

### 2. التثبيت

افتح Terminal وادخل على مجلد المشروع:

```bash
cd masar
```

ثبّت الحزم المطلوبة:

```bash
flutter pub get
```

### 3. التشغيل

شغّل التطبيق على محاكي أو جهاز:

```bash
flutter run
```

أو من Android Studio: اضغط على زر Run ▶️

### 4. البناء للنشر

لبناء APK:

```bash
flutter build apk --release
```

سيتم إنشاء الملف في: `build/app/outputs/flutter-apk/app-release.apk`

## 📝 ملاحظات مهمة

1. **البيانات الحالية ثابتة** (Static Data) - مناسبة لعرض التطبيق والتقديم في مناقشة التخرج.
2. **لربط Firebase لاحقاً:** البنية جاهزة لإضافة:
   - Firebase Authentication للمستخدمين
   - Cloud Firestore للخدمات
   - Firebase Storage للصور
